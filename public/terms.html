2. What the Service does

The Service is bridging-only software for HMRC Making Tax Digital for Income Tax. It maps period spreadsheet figures into quarterly update structures for HMRC MTD APIs for self-employment, UK property and foreign property income sources. It is not a full accounting ledger and is not affiliated with or endorsed by HMRC. It is not HMRC-recognised software and must not be used or marketed as live recognised MTD software until recognition is granted.

3. Preview vs HMRC filing

Public demo and default server configuration use a preview submit that builds the request shape but does not send data to HMRC. Live or sandbox HMRC filing only occurs when you connect HMRC OAuth with real credentials and the operator has enabled the controlled submit path. You are responsible for checking the integrity statement and your submission mode before relying on any result.

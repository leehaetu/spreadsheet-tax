Practise safely before sending anything

This account can be used in preview or HMRC sandbox mode. The current mode is always shown before a submission.

Preview or sandbox
  • Use fictional or sandbox test data
  • No production submission
  • Check the complete workflow
Production access
  • Requires HMRC production approval
  • Uses your real HMRC authorisation
  • Only enabled after explicit configuration
Who can use this account

You can use the software yourself. An agent can act only through the separate agent journey when the required HMRC authorisation is in place.

Confirm your HMRC income sources

Spreadsheet Tax retrieves the self-employment and property businesses already held by HMRC. It cannot create an HMRC business.

Requires an authorised HMRC sandbox or production connection.
Is a business missing?

Add or update the income source in your HMRC online account, then return here and retrieve the list again. Do not create a replacement inside Spreadsheet Tax.

Read HMRC’s income-source guidance

Name each HMRC source

A display name helps you recognise the source. Individual UK properties and overseas countries stay as records in your spreadsheet: HMRC treats all UK property as one business and all foreign property as one business.

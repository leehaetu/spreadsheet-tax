Security, privacy and HMRC

What happens to your file

When you use the hosted app, you upload a period spreadsheet so we can map figures for review. That is not the same as “the file never leaves your device.”

Your ongoing books remain in the spreadsheet (or system) you choose to maintain. We are bridging software, not a full ledger product.

Demo vs live HMRC

The public demo uses a safe preview submission path that does not send your update with shared live credentials. Live HMRC connection requires pilot/production controls (sign-in, OAuth, fraud-prevention headers).

See the machine-readable software integrity statement for what is real vs preview.

HMRC recognition status

Spreadsheet Tax is not HMRC-recognised software. It is not listed as recognised Making Tax Digital software for live taxpayer use. We integrate with HMRC APIs in sandbox / pilot only for development and evidence. We will not present the product as live-ready or “HMRC recognised” until recognition is actually granted.

Spreadsheet Tax is not affiliated with or endorsed by HMRC. You remain responsible for the accuracy of figures and for HMRC obligations.

What this software covers (and what it does not)

Customer product journeys (sandbox / pilot when HMRC live submit is enabled):

  • Quarterly updates — self-employment, UK property and foreign property from a spreadsheet (/app).
  • Corrections — amend an SE period after submit (/year-end).
  • Year-end steps — annual summaries, losses, other-income adjustments, tax calculation trigger/list, BSAS trigger/list/adjust, final-declaration obligations and intent-to-finalise (/year-end).
  • Practice book — client list, import, status (/workspace).

Default public mode is preview only (no HMRC HTTP). External HMRC calls need sign-in, real OAuth, and server flags. This is not HMRC-recognised software for live production filing.

Not claimed: full in-product tax bill calculation (trigger/list only; use your HMRC online account for the estimate view), EOPS, agent multi-client production authority beyond firm roles we implement, or Production APIs until HMRC grants them.

Capability evidence tags and exact sandbox HTTP outcomes: see the integrity page and operator docs. If you need software already on HMRC’s list for live use today, use HMRC’s compatible software list.

# Spreadsheet Tax

HMRC MTD ITSA **bridging-only** software for:

- Self-employed trades and businesses  
- UK and foreign property landlords  
- Bookkeepers and accountants  
- Multi-accountant practices / firms  
- Clients of those firms (client portal)  

**iOS and Android** shells: see [`mobile/README.md`](mobile/README.md).

## What you get

| Surface | URL | Purpose |
|---------|-----|---------|
| **Sales website** | `/` | Customer-facing conversion site (benefits, pricing, FAQ, firm pitch) |
| **Bridging app** | `/app` | Multi-step product: upload → review figures → submit |
| **Accountant workspace** | `/accountant` | Bookkeeper / accountant client list and import path |
| **Practice dashboard** | `/practice` | Multi-accountant firm view |
| **Client portal** | `/portal` | End-client access for firm clients |
| **Template** | `/download/template` | Free period-summary CSV |
| **License** | `/license` | Company license to sell subscriptions |

The bridging app supports **drag-and-drop upload**, **sample period files**, **income/expense cards with totals**, line-level mapping review, and HMRC submit (sandbox or confirmation preview).

## Quick start

```bash
npm install
npm run generate:test-spreadsheets
npm test
npm run test:e2e
npm start
```

**Docs / plan:** [docs/README.md](docs/README.md) · [docs/COMPLETION-PLAN.md](docs/COMPLETION-PLAN.md) · [docs/STATUS.md](docs/STATUS.md)

**Gate 0 defaults:** public submit uses safe preview (double) unless `HMRC_ALLOW_LIVE_SUBMIT=1`. Practice workflow writes are frozen unless `DEMO_PRACTICE_WRITES=1`.

Then open http://localhost:3000/

## Test spreadsheets

Real period files for manual and automated testing live in **`test-spreadsheets/`** (generated by `npm run generate:test-spreadsheets`). The app can also load them via **Try a sample** on `/app` (customer-facing labels only).

## Environment

| Variable | Description |
|----------|-------------|
| `PORT` | HTTP port (default `3000`) |
| `HMRC_CLIENT_ID` / `HMRC_CLIENT_SECRET` / `HMRC_ACCESS_TOKEN` | Optional HMRC sandbox |
| `HMRC_MODE` | `sandbox` or `double` |

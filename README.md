# Spreadsheet Tax

HMRC MTD ITSA **bridging-only** software for:

- Self-employed trades and businesses  
- UK and foreign property landlords  
- Bookkeepers and accountants  
- Multi-accountant practices / firms  
- Clients of those firms (client portal)  

**iOS and Android** shells: see [`mobile/README.md`](mobile/README.md).

## Quick start

```bash
npm install
npm run generate:test-spreadsheets
npm test
npm start
```

| Surface | URL |
|---------|-----|
| Sales | http://localhost:3000/ |
| Bridging app | http://localhost:3000/app |
| Accountant / bookkeeper | http://localhost:3000/accountant |
| Practice / firm | http://localhost:3000/practice |
| Client portal | http://localhost:3000/portal |
| Template download | http://localhost:3000/download/template |
| License | http://localhost:3000/license |

## Test spreadsheets

Real period files for manual and automated testing live in **`test-spreadsheets/`** (generated by `npm run generate:test-spreadsheets`). Import them via the bridging app.

## Environment

| Variable | Description |
|----------|-------------|
| `PORT` | HTTP port (default `3000`) |
| `HMRC_CLIENT_ID` / `HMRC_CLIENT_SECRET` / `HMRC_ACCESS_TOKEN` | Optional HMRC sandbox |
| `HMRC_MODE` | `sandbox` or `double` |
